# UnderRun Monitor

Monitoring tool for underrun events in pipes A, B, C, D.

## Information
- **Department**: Platform Labs
- **Contact**: liuty24

## Development

This project is built using [Wails](https://wails.io).

### Prerequisites
- Go 1.18+
- Node.js & npm

### Run in Dev Mode

```bash
wails dev
```

### Build

```bash
wails build
```
